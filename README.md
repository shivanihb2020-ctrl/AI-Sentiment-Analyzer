# AI Sentiment Analyzer
This project uses a **DistilBERT Transformer model** to detect if text is positive or negative.

## Features
* **Real-time analysis:** Instant results via Hugging Face.
* **Interactive UI:** Built with Google Colab and Python widgets.
* **Visualization:** Custom charts to show AI confidence scores.

## How to run
Open the `.ipynb` file in Google Colab and run all cells!
